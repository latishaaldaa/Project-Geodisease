🔧 TROUBLESHOOTING:
═══════════════════════════════════════════════════════════════════

❌ Python tidak ditemukan?
   → Install dari: https://www.python.org/downloads/
   → Pastikan "Add Python to PATH" dicentang

❌ pip tidak dikenali?
   → Restart CMD/PowerShell
   → Atau: python -m pip install -r requirements.txt

❌ Port 5000 sudah digunakan?
   → Matikan aplikasi lain yang gunakan port 5000
   → Atau ubah port di app.py baris terakhir

❌ Frontend tidak terhubung?
   → Pastikan backend sudah berjalan dulu
   → Cek console browser (F12) untuk error
   → Verifikasi URL: http://127.0.0.1:5000
